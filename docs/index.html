Snap Carousel

A lightweight vanilla JavaScript carousel library built with modern web technologies. Based on custom elements, scroll-snap, and scroll-padding.

Basic Example

Core features with smooth scrolling and autoplay 
<snap-carousel
    displayed="2"
    per-page="1"
    nav
    controls
    gap="16"
    padding="16"
    behavior="smooth"
    loop
    autoplay="3000"
    use-pause
>
</snap-carousel>
